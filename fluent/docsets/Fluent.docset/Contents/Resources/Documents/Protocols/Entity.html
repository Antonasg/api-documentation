Entity

public protocol Entity: Preparation, NodeConvertible

Represents an entity that can be stored and retrieved from the Database.

  • entity Default implementation

    The collection or table name for this entity.

    Default Implementation

    The default entity is the lowercase model pluralized.

    The name to use for internal storage.

    This should be left as the default implementation except for special cases like pivots.
