SessionsMiddleware

public final class SessionsMiddleware: Middleware

Looks for the vapor-session cookie on incoming requests and attempts to initialize a Session based on the identifier found.

If an active Session is found on the request when the response is being made, the Session identifier is returned as a vapor-session cookie.
